SOBRE O PROJETO

Feito em 12 dias, o Projeto de Xadrez foi desenvolvido por mim, Miguel Fernando, utilizando os fundamentos do paradigma da Orientação a Objetos, assunto parcialmente visto nas aulas mas que é foco dos meus estudos na linguagem de programação Java. Através da orientação dada pelo curso de Java que estou realizando, fiz o mapeamento na linguagem UML das Classes e Métodos que hoje são usados no Projeto, e durante um período de quase duas semanas programei sozinho e fiz o controle de versionamento do Projeto através da plataforma GitHub (a qual pode ser acessada abaixo) e, além disso, comentei e tentei explicar de maneira clara todas as propriedades e métodos de cada Classe específica. O projeto também foi mapeado utilizando duas camadas: a do Tabuleiro e a do Xadrez, com seus próprios métodos e delegação de lógica utilizando Encapsulamento. É um projeto simples, de visualização de terminal, mas que possui certo nível de complexidade, principalmente por respeitas os fundamentos da POO. Os tópicos principais abordados no projeto foram: Encapsulamento, Composição, Coleções, Desenvolvimento em Camadas, Padrão de Projeto, Polimorfismo, Associações, Enumerações, e Tratamento de Exceções. O jogo atualmente pode ser jogado em qualquer computador, basta seguir o passo a passo disponível no tutorial no próximo tópico.
